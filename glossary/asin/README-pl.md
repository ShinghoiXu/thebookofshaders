## asin
Zwraca arcsin parametru

### Deklaracja
```glsl
float asin(float x)
vec2 asin(vec2 x)
vec3 asin(vec3 x)
vec4 asin(vec4 x)
```

### Parametry
```x``` określa wartość, której arcsin ma zostać zwrócony.

### Opis
```asin()``` zwraca kąt, którego sinus trygonometryczny to ```x```.
