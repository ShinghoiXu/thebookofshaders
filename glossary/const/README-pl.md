## const
Kwalifikator stały

### Przykład
```glsl
const float PI = 3.14159265359;
```

### Opis
Kwalifikator ```const``` może być zastosowany do deklaracji dowolnej zmiennej, aby określić, że jej wartość nie ulegnie zmianie.
