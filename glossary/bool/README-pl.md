## bool
Typ zmiennej logicznej

### Deklaracja
```glsl
bool aBool = true;
bool bBool = bool(aInt);
bool cBool = bool(aFloat);
```

### Opis
`bool` może przyjmować dwie wartości: `true` lub `false`.
