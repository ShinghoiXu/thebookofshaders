## cos
Zwraca cosinus podanego parametru

### Deklaracja
```glsl
float cos(float angle)
vec2 cos(vec2 angle)
vec3 cos(vec3 angle)
vec4 cos(vec4 angle)
```

### Parametry
```angle``` określa wartość (w radianach), dla której ma zostać zwrócony cosinus.

### Opis
```cos()``` zwraca trygonometryczny cosinus podanego kąta.
