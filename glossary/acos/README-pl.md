## acos
Zwraca arcus cosinus podanego argumentu

### Deklaracja
```glsl
float acos(float x)  
vec2 acos(vec2 x)  
vec3 acos(vec3 x)  
vec4 acos(vec4 x)
```

### Parametry
```x``` wartość, której arcus cosinus ma zostać zwrócony.

### Opis
```acos()``` zwraca kąt, którego cosinus jest równy ```x```.
